What is Full-Stack Web Development?

Full-stack web development involves working on both the front-end and back-end of a website or web application. A "stack" refers to the combination of technologies used to create a web application, typically consisting of a front-end framework or library, a back-end framework or language, and a database.

Popular Stacks

LAMP stack: JavaScript - Linux - Apache - MySQL - PHP
LEMP stack: JavaScript - Linux - Nginx - MySQL - PHP
MEAN stack: JavaScript - MongoDB - Express - AngularJS - Node.js
Django stack: JavaScript - Python - Django - MySQL
Ruby on Rails: JavaScript - Ruby - SQLite - Rails

The MERN Stack

The MERN stack consists of MongoDB (a NoSQL database), Express.js (a web application framework for Node.js), React (a JavaScript library for building user interfaces), and Node.js (a JavaScript runtime environment for server-side scripting). Together, these technologies enable developers to build full-stack web applications using JavaScript.

Reference: MongoDB - MERN Stack Guide

Server Functions in a Web Application

In a typical web application, the server-side functions are divided into three main components:

  • Web server: Responsible for handling HTTP requests from clients (such as browsers) and serving web pages or other resources.
  • Application server: Executes the server-side code of the web application, processing requests, interacting with databases, and generating dynamic content.
  • Proxy server: Acts as an intermediary between clients and servers, forwarding requests and responses while providing additional functionalities like caching, load balancing, and security.

Reference: NGINX - Web Server vs. Application Server

Browser Request to Server

When a browser sends a request to a server, it initiates a communication process known as the request-response cycle. With Server-Side Rendering (SSR), the server processes the request, generates the HTML content, and sends it back to the browser. In Client-Side Rendering (CSR), the browser retrieves the initial HTML content and then executes JavaScript code to render and manipulate the page client-side.
